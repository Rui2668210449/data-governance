# 数据治理平台 PRD 产品需求文档

## 1. 产品概述

本平台是一套面向企业数据治理的一体化解决方案，覆盖元数据管理、数据标准、数据质量、数据安全、数据资产目录、数据生命周期管理、治理工作流与协同七大核心域，帮助企业实现"看得清、管得住、用得好"数据资产。
- 目标用户：数据治理团队、数据开发工程师、业务分析师、数据安全管理员、管理层。
- 核心价值：打通元数据-标准-质量-安全-资产闭环，提供血缘可溯、质量可度量、安全可控、资产可检索的治理能力。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 数据治理管理员 | 后台分配 | 全模块配置、策略下发、审批 |
| 数据开发工程师 | 后台分配 | 元数据采集、质量规则配置、血缘查看 |
| 数据分析师 | 后台分配 | 资产检索、血缘查询、质量看板 |
| 数据安全管理员 | 后台分配 | 分级分类、脱敏策略、审计日志 |
| 业务用户 | 后台分配 | 资产目录检索、申请使用 |

### 2.2 功能模块

1. **元数据管理**：技术/业务元数据采集、血缘解析与影响分析、元数据版本与变更审计
2. **数据标准**：标准文档管理、标准映射与落标稽核、标准版本与审批
3. **数据质量**：质量规则配置、监控调度、问题工单闭环、质量报告大屏
4. **数据安全**：数据分类分级、脱敏策略、访问审计、权限管控建议
5. **数据资产目录**：资产编目、智能检索、资产评价、个人工作台
6. **数据生命周期管理**：保留策略、归档销毁、生命周期监控
7. **治理工作流与协同**：问题工单、任务分配追踪、通知审批

## 3. 核心流程

### 3.1 元数据采集与血缘解析流程
用户配置数据源 → 采集 Worker 拉取元数据 → 解析 SQL 生成血缘 → 写入图库 → 前端渲染血缘图 → 点击节点查看详情/影响分析。

```mermaid
flowchart LR
    A["配置数据源"] --> B["采集 Worker 拉取"]
    B --> C["SQL 解析生成 AST"]
    C --> D["抽取源-目标映射"]
    D --> E["写入图库 JanusGraph"]
    E --> F["前端渲染血缘图"]
    F --> G["点击节点影响分析"]
```

### 3.2 数据质量闭环流程
配置规则 → 调度执行 Spark/Flink → 结果回写 → 触发告警 → 生成工单 → 指派责任人 → 修复复测 → 质量分更新。

```mermaid
flowchart LR
    A["配置质量规则"] --> B["调度执行"]
    B --> C{"检查通过?"}
    C -->|否| D["生成告警工单"]
    D --> E["指派责任人修复"]
    E --> F["复测验证"]
    F --> G["质量分更新"]
    C -->|是| G
```

### 3.3 资产检索与申请流程
关键词搜索 → 命中资产 → 查看详情 → 申请使用 → 审批 → 授权访问。

```mermaid
flowchart LR
    A["关键词检索"] --> B["结果列表筛选"]
    B --> C["查看资产详情"]
    C --> D["申请使用"]
    D --> E["管理员审批"]
    E --> F["授权访问"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃科技蓝 `#0052D9`（TDesign 品牌色）作为主色，辅以青蓝渐变 `#266FE8`→`#00A8E5`，背景采用 `#F5F7FA` 浅灰；暗色模式主背景 `#0F1419`。
- **强调色**：状态语义色—成功绿 `#00A870`、警告橙 `#ED7B2F`、危险红 `#E34D59`、信息蓝 `#0052D9`。
- **按钮风格**：TDesign 默认圆角 6px，主按钮采用实心填充 + 轻微投影，次按钮描边样式。
- **字体**：标题使用 `PingFang SC` / `Microsoft YaHei`，正文 14px，行高 1.5；数字/代码使用 `JetBrains Mono`。
- **布局风格**：左侧固定导航 + 顶部全局栏 + 内容区卡片式布局；血缘/审计等使用画布或宽表布局。
- **图标风格**：TDesign Icon 线性图标 + 数据治理语义化图标（表、字段、血缘、盾牌）。

### 4.3 响应式

桌面优先（≥1440px 最佳），中等屏(1280px) 自适应收缩侧栏，移动端仅保留检索与待办查看。

### 4.4 动画与交互

- 路由切换淡入淡出 200ms
- 卡片悬停轻微上浮 + 阴影加深
- 血缘节点展开有缩放过渡
- 数字指标增长使用 CountUp 动画
- 表格行展开有平滑过渡
